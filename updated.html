New Features

  • Added visual cue on the icon if page is already saved in your bookmarks.
  • @Ian - Added new shortcut for modal view. [shift]-[alt]-[d] will open add link view and [shift]-[alt]-[b] will open your bookmarks view.
  • @Jelle - Ability to change what you want to filter on in the options menu.
  • @Robert - Ability to open more than one link at a time!
  • Bookmark your current tab, change descriptions, add tags, mark private or add notes.
  • Multi-word filter your saved bookmarks by description, tag(s), date and more.
  • Modify/Delete existing bookmarks
  • Omnibox | Search from within Chrome's address bar, just type '@del+space' + keyword.
  • Shortcut access via 'shift-alt-d'
  • Context menu access, just 'right click' on the page and select 'Add to Delicious'
  • Support for Mac, Windows, Linux.
  • Customize the extension via the Options panel.
